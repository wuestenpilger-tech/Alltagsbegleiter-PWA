ALLTAGSBEGLEITER 22.6.1 – STABILITÄTSTEST

Diese Fassung baut eine APK mit einer festen privaten Testsignatur. Dadurch lassen sich spätere 22.6.x-Testversionen über diese Version installieren.

Wichtig: Da Version 22.4 mit einer anderen Signatur gebaut wurde, muss 22.4 vor der ersten Installation von 22.6.1 einmal deinstalliert werden. Danach bleiben Updates innerhalb der neuen stabilen Testreihe möglich. Vorher ggf. Daten aus 22.4 exportieren.

Build: GitHub Actions > Android APK erstellen > Workflow ausführen.
Artefakt: Alltagsbegleiter-Version-22.6.1-STABIL-APK
